# Plan - fix-exec-error-msg

## 概要

`/dev exec` 実行時に `tasks.md` が存在しない場合、現在は不明瞭なエラー（ファイルが見つからない等の汎用エラー）が発生してユーザーが混乱する。

明確なエラーメッセージと対処法を表示することで、ユーザーが次に何をすべきか（`/dev plan` を実行する等）を即座に理解できるようにする。

## 実装方針

SKILL.md の `/dev exec` ステップ1に、`tasks.md` が存在しない場合のエラーハンドリングを追記する。

具体的には以下のメッセージを表示する:

```
エラー: tasks.md が見つかりません。

原因: /dev plan が未実行、または計画が途中で中断された可能性があります。

対処法:
  1. /dev plan を実行して計画ファイルを生成してください。
  2. すでに plan.md のみ存在する場合は、/dev plan のステップ6（アウトプット生成）から再開してください。
```

## 影響範囲

| ファイル | 変更種別 | 理由 |
|---------|---------|------|
| `plugins/dev/skills/dev/SKILL.md` | 修正 | `/dev exec` ステップ1のフロー記述にエラーハンドリングを追加 |

## 考慮事項

- **後方互換性**: SKILL.md はAIへの指示書のため、追記のみで既存の動作を壊さない
- **スコープ**: `plan.md` が存在しない場合も同様の問題があるが、今回は `tasks.md` のみに絞る
- **メッセージの明確性**: エラーの原因・対処法の両方を含めてユーザーが自己解決できるようにする

## トレーサビリティ

| 要件 | コンポーネント | API | データ |
|------|--------------|-----|--------|
| tasks.md 不在時に明確なエラーを表示 | SKILL.md（/dev exec ステップ1） | - | tasks.md |
